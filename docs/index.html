À propos du jeu

Une course contre la montre dans une ville en ébullition

Delivery Rush est un jeu de livraison multijoueur vue du dessus, construit avec Python et Pygame. Incarnez un livreur intrépide au volant d'un véhicule personnalisé, naviguez dans une immense carte urbaine et complétez des missions de livraison avant vos adversaires.

Le jeu propose une physique de conduite réaliste avec accélération, freinage et dérapage, un système de réseau UDP pour le jeu en ligne, et une carte scrollable de 12 000 × 12 000 pixels créée avec l'éditeur Tiled.

Comment jouer

Prêt à démarrer en 3 étapes

01
🐍

Installer les dépendances

Assurez-vous d'avoir Python 3.8+ installé, puis installez les dépendances du projet :

bash 
pip install -r requirements.txt
02
🖥️

Lancer le serveur

Démarrez le serveur UDP qui gérera les connexions multijoueurs :

bash 
python server.py
03
🎮

Lancer le jeu

Lancez le client principal et commencez à jouer :

bash 
python main.py

Configuration système

Ce dont vous avez besoin

🐍

Python

3.8 ou supérieur

Requis pour exécuter le jeu. Téléchargez Python sur python.org

🎮

Pygame

2.0+

Bibliothèque de développement de jeux 2D pour Python. Installée automatiquement via pip.

🗺️

PyTMX & PyScroll

Dernière version

Pour le chargement des cartes Tiled et le rendu du scrolling de la carte du jeu.

🌐

Réseau

LAN / Internet

Connexion réseau nécessaire pour le mode multijoueur. Mode solo disponible localement.

💡

Toutes les dépendances Python sont listées dans requirements.txt et peuvent être installées en une seule commande : pip install -r requirements.txt
